# Legado Crítico: API de Pedidos e Logística

Este projeto simula uma API Node.js com Express + TypeScript para gestão de pedidos e logística de e-commerce.

Ele foi criado intencionalmente como um **legado crítico**, com alta dívida técnica, baixa confiabilidade e necessidade urgente de refatoração.

## Executar

```bash
npm install
npm start
```
